SuperMap Online Tutorial

Introduction to JavaScript API Overview

SuperMap Online JavaScript APIs support multiple data and map sources, multiple terminals and browsers. You can use them to create a map applications with rich functions and high interactiveness.

SuperMap Online JavaScript APIs has two parts: iClient for JavaScript and iConnector.

SuperMap iClient for JavaScript

It is a package for developing a GIS client app using JavaScript. It supports multiple data and map sources, multiple terminals and browsers for developing a map applications with rich functions and high interactiveness.

SuperMap iConnector

It is an open source tool to connect SuperMap GIS services with third party JavaScript map applications. Using iConnector, you can reuse your existing GIS code and integrate professional SuperMap REST GIS services.

Development Authorization (Apply for a Key)

To use SuperMap Online services and APIs to develop your own Web application, you need to apply a key first. A key is a token with a combination of random letters and numbers.

A key can authorize you to use the following two types of services:

  • Basic GIS service. Each key allows: 5000 free requests per day.

SuperMap Online supports GIS services including routing, geocoding, traffic transfer, etc. It is free for use. You can apply for a key and use these services.

  • After you upload your data and publish your REST map service, you have up to 5000 free requests per day for each key (one map tile is assumed as a request).

A key is required whenever you use your REST map service that you published using your own data. You need to generate a key before you develop your application.

For details, please see: Development Authorization.
