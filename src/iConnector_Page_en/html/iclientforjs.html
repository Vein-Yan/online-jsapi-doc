SuperMap iClient for JavaScript

Product Introduction

SuperMap iClient for JavaScript is a Web GIS application development package for HTML5. It supports multiple sources of data, multiple terminals and browsers. It has the following features:

  • It enables a flexible interactive design and rich representation of data. The HTML based Canvas allows for high efficient and stable displaying of map tiles.
  • It supports data and maps from OGC-compliant services, SuperMap iServer services, SuperMap Cloud, and popular Web map providers including Baidu Maps, Google Maps, and Tianditu.
  • Based on the PhoneGap open source development framework, SuperMap iClient for JavaScript supports offline map cache via plug-in, and packaging your map application to one that can run on Android devices directly.
  • Support large data interaction, rendering and visualization. The UTFGrid layer provides the solutions for the large data interaction.
  • It provides rich visualization effects, including vector map tiles, thematic mapping on the client, and displaying spatial-temporal data.

Main Features

Map Service

SuperMap iClient for JavaScript supports multiple map services, including:

  • SuperMap Cloud Maps
  • SuperMap iServer tiled cache map
  • SuperMap iServer tiled dynamic map
  • Vector feature layers (VML, SVG, Canvas, and Canvas2）
  • Maker layers (Marker)
  • Elements layers (adding html elements to maps)
  • Aggregation layer
  • Heat grid layer
  • Heat map layer
  • Front end thematic map layer (including label map, unique value map, ranges map, and graph map)
  • Front end vector feature rendering layer (CartoCSS can be used to change map styles)
  • UTFGrid layer (used to quicly respond to interactions and providing attribution)
  • OGC service layer (WMS, WMTS, etc)
  • Spatial-temporal layer (to enable a dynamic effect on the map)
Map Operations

Besides the basic map operations like pan, zoom, layer control, etc, more operations are supported:

  • Overview map operation
  • Display of Scale bar
  • Drawing, editing, seleting and draging vector features
  • Selecting points on GOIs layer
  • Interacting with UTFGrid layer
  • Rolling Blinds
  • Positioning
  • Selecting aggregation
  • Display of mouse position
  • Display of copyright info
Measurements
  • Distance measurement
  • Area measurement

GIS Analysis Service

SuperMap iClient for JavaScript provides a set of easy-to-use APIs for developers to use and overlay GIS Analyst services published by SuperMap iServer. SuperMap iClient for JavaScript supports a complete set of GIS functions, including measurement, query, spatial Analyst, network Analyst, bus transfer Analyst, etc.
