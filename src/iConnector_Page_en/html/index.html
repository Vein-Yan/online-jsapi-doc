What is iConnector

Scenario:

  • You have developed a GIS application (e.g. a map application);
  • and you want to integrate GIS functions, e.g. thematic mapping, spatial analysis, from SuperMap GIS services.
  • The ideal approach is to make full use of your existing code from your application, without starting from scratch.

In this case, you can use SuperMap iConnector.

It is an open source tool to connect SuperMap GIS services with third party JavaScript map applications. Using iConnector, you can reuse your existing GIS code and integrate professional SuperMap REST GIS services. These GIS services will be added to your GIS applications.

The open source of iConnector in GitHub: https://github.com/SuperMap/iConnector

SuperMap online JS libraryhttp://www.supermapol.com/resources/api/libs/SuperMap.Include.js

Main Functions of iConnector

Matrix of Functions
  1. Add a SuperMap iServer map to a base map from third-party JavaScript API
  2. Add a SuperMap thematic map to a base map from third-party JavaScript API
  3. Add SuperMap geometries to a base map from third-party JavaScript API
  4. Add SuperMap iServer spatial analysis results to a base map from third-party JavaScript API
